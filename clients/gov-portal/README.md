# INDIS — Government Portal

> Ministry integration dashboard (PRD §FR-009)

## Authentication
- Certificate-based (no password-only access)
- Role-based data visibility (data minimization)
- Multi-level approval chains per credential type

## Technology
- GraphQL for complex data queries
- React + TypeScript frontend
- RTL-first Persian interface

## Modules
- Ministry Integration Dashboard (FR-009)
- Electoral Authority Module (FR-010)
- Transitional Justice Module (FR-011)
- Enrollment Agent Management
- Audit Log Viewer (read-only for auditors)
